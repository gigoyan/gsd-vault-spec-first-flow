# Project Idea Document Template

Use this template for the current project's Project Idea Document.
Do not use it to replace the bootstrap charter in `PROJECT.md`.
When intake is document-first or continuing from partial maturity, extract known inputs into this artifact before re-asking the user and keep unresolved items explicit as `Unknown` or `Suggested`.

## Document Purpose
- Capture the product or workflow opportunity before detailed implementation planning starts.
- Establish enough clarity to decide whether the idea is worth specifying, scoping, and implementing.
- Link this document to the governing charter, roadmap, and later Technical Specification.
